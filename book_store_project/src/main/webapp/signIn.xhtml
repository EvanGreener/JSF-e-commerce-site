Login Information
When logging in, if an email that the user enters is not in the database, an error message will be displayed. If the email or password fields are left blank after changing them, respective error messages will be displayed. If a password associated with an existing email is incorrect, the user will only be made aware after attempting to log in, after which they will be asked to enter the correct password. In the case where a user has not yet made an account, they may click the "Register" link located below the login form to create an account.
